New Cubee.

With Just Cubees there is a small limited list of Cubees that can be used to build models.

Once selected the new Cubee appears at the top of the room so that it is clearly visible. You can then move it to which ever position you wish.

Moving the Cubee is done with the cursor keys along with < and >.
Alternatively use the coloured section on the menu bar. This also shows which colour the keys move the Cubee towards. White represents the ceiling, which remains unseen.

Rotating the Cubee is within the Selection Menu.
