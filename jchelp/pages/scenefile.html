File

File Sub Menu.

New
Deletes the current scene and starts a new one.

Fetch Model.
Fetches a model from the Store and add it to the scene.

Open Model.
Open a model from the local storage and add it to the scene.

Import Model.
Browse to an exported model file and import into the scene room and add it to the scene.

Export Scene.
Opens a new browser window where the data describing the scene is written. Using browser tools save this data to a file on the local computer. The saved file be a txt or a babylon file.
The file may be used in your own babylon.js projects. If you use textures then you need to obtain copies of the appropriate texture images.

Download Scene.
Possibly easier than Export to the same effect. Give the scene a filename and it is downloaded to your browser's download folder.
Repeated filenames are automatically renumbered.

Import Scene.
Browse to an exported or downloaded scene file and import into the scene room. This will delete the current scene.
