Using Extrude.

The extrusion is controlled by using the square and circular markers.

The top marker on the axis line can only move on the axis.
The bottom marker on the axis line is fixed.

The cross section of the extrusion is controlled by the tool on the right.
The tool at the bottom controls further parameters along the length of the extrusion.

Cross Section Tool

The square and circular markers control the shape of the cross section.
Making changes to the cross section that puts the centre white dot outside the cross section will produce
strange results at the start and end of the extrusion and is best avoided.

Further adjustments to the cross section can be made by adding new markers.
Click anywhere on the outline of the cross section and a dialogue box opens, so you can do this.


This dialogue box can also be used to straighten or curve sections and to delete them.

Parameter Tool


The scale, angle and position of the cross section can be changed along the length of the extrusion as it is extruded.
The values at any point are determined by the position of the parameter line at any point.
Switch between variables using the variable dialogue box.
Rotation is in the plane shown, ie about the z axis.
Tilt is rotating the top towards or away from you; ie about the x axis.
Twist is rotating towards the left or right; ie about the y axis.

Locking the scale or position always locks the scale or movement to the current x scale and x position.

Additional markers can be added to the parameter line and sections of the line
can be straightened or curved by clicking on the line and using the dialogue box
drag the markers to adjust the line.


As each marker is selected its position along the line is labelled at the top and its value is labelled on the left.

A label for the maximum, minimum and middles values are also shown.

Clicking on a label brings up a dialogue box that allows you to change the values
for that label and adjust the extrusion accordingley.
Note that scales are always kept between 0 and 1, with 1 being the size of cross section as constructed.
The maximum length of the extrusion is fixed.
