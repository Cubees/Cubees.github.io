File

There are two ways to save the turned shape on your local computer.

Export.
Opens a new browser window where the data describing the model is written. Using browser tools save this data to a file on the local computer. The saved file be a txt or a babylon file.
The file may be used in your own babylon.js projects. If you use textures then you need to obtain copies of the appropriate texture images.

Download
This allows you to name your file and download it into your download folder for the browser.
The file will have the extension .babylon

Use the Save option when the download dialogue box opens.
The download file is text and you can load it into Notepad to see the text.
